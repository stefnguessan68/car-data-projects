# Car Data Projects

This repository contains beginner data analysis projects focused on motorsport and racing data, created while learning Python, Jupyter Notebook, and data science fundamentals.

The goal of these projects is to explore performance metrics, analyze relationships within racing datasets, and communicate insights clearly through structured analysis.

---

## Projects

### 1. F1 Race Outcome Hypothesis Testing

This project explores which driver-centric and team-centric variables are most closely related to finishing position using a synthetic Formula 1 dataset.

The analysis is structured around testing simple hypotheses and evaluating relationships between variables and race outcomes.

Skills practiced:

* data inspection and cleaning
* grouping and aggregation
* feature engineering (finish position grouping)
* data visualization (scatter plots, boxplots)
* analytical reasoning and hypothesis testing

Key question(s):

* Which driver metrics (grid position, tire wear, lap time) relate most to race outcomes?
* Which team metrics (car speed, pit crew efficiency, downforce) show the strongest relationship with finishing position?
* Do team factors appear more influential than driver-specific metrics?
